# AI Resume Analyzer & Job Match Dashboard

A React + TypeScript app that compares a resume against a job description, calculates an ATS-style match score, finds missing keywords, and visualizes skill gaps.

## Features

- Resume and job description analyzer
- ATS-style match score
- Missing keyword detection
- Skill category breakdown
- Improvement suggestions
- Saved analysis dashboard
- Responsive modern UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand
- Recharts
